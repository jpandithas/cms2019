# cms2019
The 2019 version of the CMS for the Engineering Courses of NYC

## Notice: 
Download the database of the project

## Idea
The idea behind this project is to create an easy to build CMS that is based on an MVC pattern 

## Codebase
Object Oriented PHP

## Features 
The following features should be supported:
- [x] PHP OOP Framework
- [x] Boostraping capability 
- [x] Router to support the Controller 
- [x] Modular Design 
- [x] Themes (Responsive)
- [x] Database Connector Class 
- [x] User Roles
- [x] Permissions System
- [x] Manage Permission
- [x] Home Page, 404 and 403 Pages
- [x] Query ORM 
- Queries:
  - [x] Insert 
  - [x] Delete
  - [x] Update 
  - [x] Select 
  - [x] Where Clause 
  - [x] And Clause
  - [x] Or Clause 
  - [x] Limit Clause 
  - [x] Order By
- Webform Builder
  - [x] Radio Button
  - [x] Checkbox 
  - [x] Text Field
  - [x] Password Field
  - [x] Text Area
  - [x] Submit Button
  - [x] Select Menu 
  - [x] Text to Accompany Form 
  - [x] Field Separator
